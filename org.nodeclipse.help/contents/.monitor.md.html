Run/Debug with Monitor

https://github.com/Nodeclipse/nodeclipse-1/issues/57 running app.js with node-dev, forever, supervisor, nodemon etc

Configure your source file monitor of your choice:

forever [action] [options] SCRIPT [script-options] nodemon [your node app] node-dev foo.js supervisor [options] <program>

forever - 2700 stars node-dev https://github.com/fgnass/node-dev - 380 stars (doesn't require any configuration) nodemon 1500 stars - support running .coffee supervisor 1100 stars
